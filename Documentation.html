Part I: The Philosophy

"What were the rules I used to write this for authors?"

1. The Minimum Needed to Write It was a massive task to boil down everything writing software allows you to do into just what an author needs to do. I pared the fat down to the bone. If it doesn't put words on the page, it’s gone.

2. The Cuts (Distractions)

Multiple Fonts: There isn't a need to use multiple fonts in a manuscript. If you need to indicate a computer speaking, use Bold or Italics. Fonts are for professional layout, not for drafting.

Checkboxes & Lists: This isn't a To-Do list application. If you need a checklist, use a notebook.

Colors: Nope. No text colors to play with. No color-coding the tree.

The Corkboard: Sigh. I love that feature in other apps. But it’s a distraction, and it is far too easy to spend hours shuffling index cards when you should be writing scenes.

Version Control: If you are writing a draft, you know that. If you are writing a correction, you know that too. The software doesn't manage "snapshots" because that’s just another form of procrastination.

3. The Essentials (The Tools)

Formatting: Bold, Italic, Underline, and Alignment (Left/Right/Center/Justify) are in there. That’s it.

Structure: A robust Tree structure to keep track of your chapters and scenes? In there.

Focus: Full-screen, distraction-free editing? In there.

Environment: Dark/Light/System palette configuration? In there. For those of us with difficulty seeing, you can pick the font and character size for the whole novel. You need to be comfortable to write.

Safety: Background saving is standard. You don't have to stop typing to save; it happens silently so you never lose your work if the power goes out.

4. The "Fancy" Features (Allowed only because they help flow)

Text-to-Speech: It will read to you if you need it while editing. This is my only nod to "fancy" features because I use it religiously to catch rhythm errors.

Spell Check: Available, but only when you ask for it. No distracting red squiggles while you are trying to be creative.

Typing Sounds: Optional (for those who need the clack-clack of a typewriter to feel productive).

5. The Keyboard Rule Everything the program does is aimed at one thing: Flow. Everything you can do with the mouse is available at the keyboard so you can keep your eyes on the screen and your mind on the story. No bells, bings, or flashes unless you want them (and the flashes are a hard "No," sorry).

6. The Hidden Power (Scripting) The program is simple on the outside, but it hides powerful customization. There is a scripting language built in. It’s not Python or Lua; it’s a language written by me to be intuitive, stack-based, and simple.

The Parity Principle: If you can do it with the mouse or keyboard, a script can do it, too.

The Restriction: You cannot edit scripts inside this program. That’s coding, not writing. If you need a tool, write the script in a text editor and import it. This keeps the "Novel Writer" separate from the "Tool Builder."

When you first start TextSmith, it's ready for you to write. You have a blank slate. The first time you save the novel, TextSmith will ask you what to name it and where to put it (it defaults to your Documents directory).

On the left, you will see an item named <unnamed_0>. This is the start of your novel. You can either write your whole book right here, or create new items to hold your writing "underneath" it (or inside it, if you prefer).

To create an item under (or inside) another item, just hit Alt+N. You'll see a dialog asking you to name the item and create tags for it. Give it a name (like 'Prologue') and hit Enter. We will talk more about tags later; right now, you need to get writing.

You can hit Alt+N as many times as you need to lay out the structure of your novel.

Moving Around:

You can click on any item in the tree to switch to it.

The Keyboard Way: If you are writing and don't want to reach for the mouse, hit Shift+Tab to jump over to the tree, and use your arrow keys to navigate to the item you want. Hit Tab to jump back into the editor and start typing.

TextSmith remembers exactly where you left off in each item.

Saving: Hit Ctrl+S to save the novel, give it a name, and get back to work. TextSmith also autosaves as you go—saving quietly in the background at the intervals you tell it to—so you don't need to worry about losing your work.

The Menus

There are five main menus (File, Edit, Novel, View, and Help). Let's go through them in order and describe what each one does.

A quick note: TextSmith goes to great lengths to only enable menu selections that make sense at any given time. This reduces the classic "what-just-happened?" problem by preventing you from clicking things you shouldn't.

The File Menu

The File menu is where all things file-related go.

New (Ctrl+N): Start a new novel. If your current novel has unsaved changes, TextSmith will ask if you want to save them, discard them, or cancel the command. (The default action is Save, of course!)

Open (Ctrl+O): Open an existing novel. Just like 'New', it will ask you to save any current changes so you don't unknowingly lose your work. The document will open exactly where you left off when you last saved it.

Save (Ctrl+S): Saves your work to the current file. If you've never saved it before, it will ask you for a filename. You can cancel if you change your mind, but TextSmith will remember that the document is still unsaved.

Save As (Ctrl+Shift+S): Saves the novel under a brand new name.

Export (Ctrl+E ...: Export the current novel into one of the following formats: EPub (...Ctrl+E), HTML (...Ctrl+H), Markdown (...Ctrl+M), PDF (...Ctrl+P), or RTF (...Ctrl+R, most word processors like Microsoft Word and LibreOffice Writer can read RTF files).

Page Setup (Ctrl+Shift+P): Set up your margins, headers, and footers for printing your documents.

Print (Ctrl+P): Send the novel to the printer.

Exit (Alt+F4): Leave the program. If you have unsaved changes, TextSmith will step in to keep you from losing any work before it closes.

The Edit Menu

The Edit menu is where you find typical—and sometimes not-so-typical—text editing tools.

Undo (Ctrl+Z): Undo the last thing you did. The program keeps track of your actions like a stack of plates, letting you undo them in the exact reverse order you did them.

Redo (Ctrl+Y): If you undid something you wish you hadn't (perhaps from an overzealous string of Ctrl+Z keypresses), Redo lets you walk that back, putting things back on the "stack of plates"—as long as you haven't typed anything new yet.

Cut (Ctrl+X): Take the highlighted text, copy it to your paste buffer, and delete it from the editor.

Copy (Ctrl+C): Same as Cut, except the highlighted text is left exactly where it is.

Paste (Ctrl+V): Takes whatever is in your paste buffer (from anywhere on your computer) and plops it into TextSmith. This may involve stripping out a lot of formatting fluff if it's coming from a complex webpage or program. TextSmith does have limited image handling, so give it a try!

Find & Replace (Ctrl+F): This reveals a search bar at the bottom of the display. Type what you want to find and press Return. You can easily replace the found text with whatever you type in the replace field, or even replace all occurrences at once.

Find Next (F3): If you've already searched for something, this jumps directly to the next occurrence.

Font: Not exactly what you think! This simply lets you toggle Bold (Ctrl+B), Italic (Ctrl+I), and Underline (Ctrl+U) on your selected text.

Paragraph (Ctrl+J ...): Align your text (Left (...Ctrl+L) , Center (...Ctrl+C), Right (...Ctrl+R), Justify (...Ctrl+J) ) or change the indentation (Indent, Unindent). Note: Left is sometimes called "ragged right", and Right is sometimes called "ragged left", just to confuse things.

Lowercase (Ctrl+L): Change the selected text to entirely lowercase.

Uppercase (Alt+L): Change the selected text to entirely uppercase.

Preferences (Ctrl+Shirt+R): Lets you set your default program settings. This includes the editor font, autosave intervals, text-to-speech voices, and more. (More on preferences later in the tour!).

The Novel Menu

This menu handles the structure and organization of your novel's items.

Add Item (Alt+N): Adds a new item underneath your current one, allowing you to name it and assign tags.

Edit Item (Alt+E): Allows you to change the name and tags of an existing item.

Remove Item (Alt+Del): Deletes an item. Don't worry, you can use Undo if you make a mistake. You are welcome!

Move Current Item Up (Alt+Up): Moves the current item higher up in the document tree. If it reaches the top of its current parent group, it will "pop out" and move above its parent.

Move Current Item Down (Alt+Down): Moves the item down the tree, or out of its parent group if it reaches the bottom.

Move Current Item Out (Alt+Left): For when your nested lists get too deep, this simply promotes the item directly out of its current parent folder.

Open/Close Current Item (Alt+O): Expands the current item to see its children, or collapses it to hide them.

Open All (Alt+A): Expands every single item in your document tree. It can be nice to see the whole structure, or it might just be overwhelming.

Close All (Alt+C): If "Open All" was too much, this collapses everything. Take a deep breath and start opening things one at a time.

The View Menu

This menu controls how you look at your writing environment.

Distraction Free (F11): This brings up the distraction-free window. I develop TextSmith on a 4K monitor (actually, it's an old TV I got for free, which is great for vision issues), so my screen has a lot of empty space. Distraction-free mode dynamically adjusts to your monitor to give you a clean slate. You can drag the margins to make it look exactly like a page, and it completely blocks out the main TextSmith window until you hit ESC or F11 to exit.

Features in this mode: You can type, use basic text editing (cut, copy, paste, undo, redo, formatting), and check your word counts at the bottom of the left-hand bar. If you've enabled typing sounds in your preferences, you'll hear them here.

Read To Me (Ctrl+Shift+R): Starts reading the current item out loud using your operating system's selected text-to-speech voice. (If your OS doesn't have text-to-speech installed, it won't work—I can't program that part... yet!)

Word Counts: Displays your current word counts clearly in the left part of the bottom status bar so you can track your progress.
